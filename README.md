# 🛡️ SafeTravel AI – Privacy-Preserving Travel Safety App

## 📌 Overview
SafeTravel AI is a smart travel safety application designed for solo, group, and female travelers. It focuses on providing **real-time risk prediction**, **privacy protection**, and **offline safety support** to help users navigate unfamiliar places safely.

---

## 🚨 Problem Statement
Existing safety apps mainly provide reactive solutions like SOS alerts. They lack:
- Real-time risk prediction
- Offline functionality
- Privacy-preserving data handling

This project aims to solve these gaps by creating a **proactive safety system**.

---

## 💡 Solution
SafeTravel AI integrates:
- 📊 Risk prediction system
- 🔐 Privacy-preserving architecture
- 📡 Offline safety support
- 📍 Location-based alerts

---

## ✨ Features

- 🔐 **User Authentication**
  - OTP-based login using Firebase

- 📍 **Live Location Tracking**
  - Real-time GPS tracking with maps

- ⚠️ **Safety Score System**
  - Calculates risk based on:
    - Time (day/night)
    - Connectivity
    - User reports (future scope)

- 🚨 **SOS Alert System**
  - Sends emergency alert with live location

- 📡 **Offline Support (Concept)**
  - Stores safety data locally for low-network areas

- 🧠 **AI-Based Risk Prediction (Future Scope)**
  - Uses ML models to predict safety levels
